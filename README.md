## Project Summary

Campus Connect is a Flask web application that provides a simple **CRUD (Create, Read, Update, Delete)** interface for managing **community resources and civic engagement events** such as food pantries, legal aid programs, protests, and student support services.

This project demonstrates:
- Flask routing and blueprints  
- Jinja2 templates and dynamic rendering  
- Database integration with MySQL via the `cs304dbi` library  
- Form handling and flash messages  
- Static styling with CSS
